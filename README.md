# OpenTalk — Live Comments Application

## How to run locally

**Requirements:** Node.js 18+ or Bun

```bash
# with bun (recommended)
bun install
bun dev

# or with npm
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

To build for production:

```bash
bun run build
# or
npm run build
```
